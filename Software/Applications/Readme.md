1. Download [OpenFrameworks](http://openframeworks.cc/download/)
1. Move the ZooidManager folder in your OpenFrameworks/apps/myApps/ folder
1. You can then edit the code, add new fewatures, compile and run from XCode or Visual Studio.
1. To create new client applications, use OF's projectGenerator and include the ofxNetwork addon. 
1. Copy rapidjson and src folders from API/OpenFrameworks/ into your application folder, add #include "ZooidManager.h" in ofApp.h
